# Research Process

## Objective
Standardize discovery before any layout or component generation.

## Sequence
1. Clarify project scope and constraints.
2. Map business goals to measurable outcomes.
3. Define primary/secondary user segments.
4. Analyze competitor experiences and gaps.
5. Identify high-value user journeys.
6. Produce UI direction brief.

## Deliverables
- Business goal summary
- User and context summary
- Competitor findings
- Journey map highlights
- UX opportunity list
- Design direction recommendation
