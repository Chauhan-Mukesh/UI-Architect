# Checklist — Documentation

## Findability
- [ ] Search present and functional
- [ ] Navigation reflects real IA, not marketing sections
- [ ] Each page has a stable URL and permalink to headings
- [ ] Cross-links between related topics

## Content
- [ ] Every page has title, one-line summary, last-updated date
- [ ] Code samples copyable and syntax-highlighted
- [ ] Prev/next links at page bottom
- [ ] Version switcher present when versions exist

## Accessibility
- [ ] Heading order valid; anchor links visible on hover
- [ ] Code blocks use `<pre><code>` with language classes
- [ ] Dark mode at parity with light
- [ ] Table of contents (if present) uses `<nav aria-label="On this page">`

## Performance
- [ ] Search index reasonable in size
- [ ] No render-blocking scripts
- [ ] Client-side syntax highlighting deferred where possible
