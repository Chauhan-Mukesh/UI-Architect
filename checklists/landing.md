# Checklist — Landing Page

## Above the fold
- [ ] Answers "am I in the right place?" in ≤ 5 seconds
- [ ] Single primary CTA visible
- [ ] Trust signal present (logo strip, rating, quote) if genuine
- [ ] Works on 360×640 without CTA cropped

## Structure
- [ ] Section sequence follows the visitor's decision path
- [ ] No section repeats the same claim
- [ ] Trust and objection-handling appear before final CTA
- [ ] FAQ addresses real objections, not marketing bullets

## Content
- [ ] Headline is a testable sentence, not a slogan
- [ ] All CTAs verb+object
- [ ] Testimonials attributed with name, role, company
- [ ] Numbers with unit, timeframe, and source

## Accessibility
- [ ] WCAG 2.2 AA contrast on every combination
- [ ] Keyboard traversal complete
- [ ] Reduced-motion respected
- [ ] Alt text on every image (or explicit `alt=""`)

## Performance
- [ ] LCP ≤ 2.5s (hero optimized, preloaded)
- [ ] CLS ≤ 0.1 (media has dimensions)
- [ ] JS < 100 KB compressed on the initial payload
