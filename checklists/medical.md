# Checklist — Medical / Healthcare

## Trust
- [ ] Credentials and certifications visible
- [ ] Provider names, roles, and photos are real
- [ ] Contact and emergency information immediate
- [ ] Compliance signals (HIPAA, GDPR, accessibility) where relevant

## Content
- [ ] Language is plain (target 8th-grade reading level unless clinical)
- [ ] Appointment or intake path is one action
- [ ] Insurance / cost information visible or clearly linked
- [ ] Emergency contact prominent

## Accessibility
- [ ] Larger base type size (17–18px)
- [ ] Contrast well above AA on critical information
- [ ] Forms accessible; error handling meets `accessibility/forms.md`
- [ ] Motion restrained; reduced-motion respected

## Performance
- [ ] Fast load on low-bandwidth conditions
- [ ] Images optimized; hero not blocking LCP
- [ ] Third-party scripts audited (chat widgets, analytics, ad pixels)
