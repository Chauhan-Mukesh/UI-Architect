# Checklist — Portfolio

## Positioning
- [ ] "What this person does" clear in 5 seconds
- [ ] Selected work count is 3–8, not 20+
- [ ] Contact action is singular and obvious
- [ ] Case studies include role, problem, outcome, timeframe

## Presentation
- [ ] Thumbnails reflect the finished work
- [ ] Layout breaks the grid intentionally where appropriate
- [ ] Imagery gets layout weight; no lorem-styled placeholders
- [ ] Case study depth carries the pitch; index page stays quiet

## Accessibility
- [ ] Thumbnail links have accessible names including title + role
- [ ] Focus rings visible against varied imagery
- [ ] Motion is respectful of reduced-motion preference
- [ ] Alt text for imagery is meaningful, not filename

## Performance
- [ ] Case study images optimized (AVIF/WebP)
- [ ] LCP under 2.5s on the index
- [ ] Custom fonts subset and preloaded
