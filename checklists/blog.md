# Checklist — Blog

## Index
- [ ] Post cards include title, date, author, category, read time
- [ ] Pagination or search available
- [ ] Categories or tags navigable
- [ ] RSS/atom feed provided

## Post
- [ ] Reading measure 60–75ch
- [ ] Images have meaningful alt
- [ ] Related posts at the bottom
- [ ] Share affordance if audience benefits

## Accessibility
- [ ] Each post is `<article>`; dates use `<time datetime>`
- [ ] Blockquotes use `<blockquote>` with attribution
- [ ] Sticky elements don't obscure content
- [ ] Newsletter modal doesn't ambush on first visit

## SEO
- [ ] Unique title and description per post
- [ ] `Article` / `BlogPosting` structured data
- [ ] Canonical URL set
- [ ] Author page linked (if author has a page)
