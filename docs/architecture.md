# UI Architect Architecture

This repository stores domain and implementation knowledge modules consumed by `SKILL.md` workflow orchestration.

## Layers
1. Research and domain understanding (`research/`, `industries/`)
2. Design strategy (`knowledge/`, `layouts/`, `components/`, `typography/`, `colors/`)
3. Implementation standards (`html/`, `css/`, `javascript/`)
4. Quality gates (`accessibility/`, `performance/`, `seo/`, `validators/`, `self-review/`)
5. Reusable execution inputs (`prompts/`, `templates/`, `examples/`, `checklists/`)
