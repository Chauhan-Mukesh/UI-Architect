# Template — Marketing Site

## Purpose
Structural spec for a multi-page marketing site (home, product, pricing, about, contact, legal).

## Standard pages
1. Home — see `templates/landing-page/`
2. Product / features — deeper dives per capability
3. Pricing — see `layouts/pricing/`
4. About / team / mission
5. Customers / case studies
6. Blog / resources — see `templates/blog/`
7. Contact
8. Legal (privacy, terms, cookies)

## Cross-references
- Layouts: hero, landing, pricing, footer.
- Components: navbar, footer, cards, forms, testimonials, buttons, faq.
- Quality: checklists as applicable, `self-review/questions.md`.

## Rules
- Consistent nav and footer across every page.
- Design tokens shared everywhere.
- Every page has a purpose and a single primary action.

## Anti-patterns
- Different nav on different pages.
- Legal pages that break the design system.
- Contact page as a form with no other route to reach the company.
