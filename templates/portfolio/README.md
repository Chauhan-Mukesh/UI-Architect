# Template — Portfolio

## Purpose
Structural spec for a personal or studio portfolio.

## Sections
1. Introduction (positioning line, discipline, current status)
2. Selected work (3–8 items — see `layouts/portfolio/`)
3. About / voice
4. Contact (singular action)

## Optional
- Individual case-study detail pages
- Now / currently module
- Press or awards strip (if it supports positioning)

## Cross-references
- Layout: `layouts/portfolio/variants/01|02|03`.
- Components: cards, gallery, testimonials (rarely).
- Quality: `checklists/portfolio.md`.

## Rules
- Curate ruthlessly.
- Case study depth carries the pitch; index stays quiet.
- One contact action.

## Anti-patterns
- Displaying every project ever done.
- Fake case-study "outcomes" without evidence.
- Contact form + Calendly + email all competing.
