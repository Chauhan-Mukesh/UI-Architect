# Example — Restaurant

## Prompt
"Website for a neighborhood Italian restaurant."

## Research
- Intents: reserve, view menu, find location/hours.
- Persona: local resident + occasional visitor from adjacent neighborhoods.
- Journey risks: menu locked in PDF, stale hours, no click-to-call on mobile.

## Design direction
- Tone: warm, personal, unpretentious.
- Layout: single-page with quick access to all three intents.
- Type: `editorial` voice softened — serif display + humanist sans body.
- Colors: cream + charcoal + terracotta accent.
- Motion: minimal — one hero image, no auto-video.

## Output plan
- Hero: signature dish photo + name + Reserve CTA.
- Facts strip: hours, address, phone.
- Menu section on-page (structured HTML, not PDF).
- Reservation widget or clear reservation link.
- Story / owner / chef paragraph with a real photo.
- Gallery: 6–9 images (room, food, moments).
- Location: map + directions link + click-to-call.

## Review notes
- Mobile reserve possible in under 15 seconds.
- `Restaurant` structured data with `OpeningHours` and address.
- Hours are current and easy to update.
- Photos are original.
