# Example — Finance

## Prompt
"Landing page for a robo-advisor targeting first-time investors."

## Research
- Persona: hesitant first-time investor, cautious about fees and risk.
- Journey risks: trust, cost clarity, jargon fatigue.
- Compliance considerations: risk disclosures, non-guaranteed-return language.

## Design direction
- Tone: calm, honest, reassuring.
- Layout: landing variant 02 (story-driven).
- Type: humanist sans; larger base for readability.
- Colors: `industries/finance/recommended-colors.md` — deep blue-green + off-white + measured accent.
- Motion: minimal, purposeful.

## Output plan
- Hero framing the tension: investing feels intimidating.
- Story: how it works in one sentence, then plain-language mechanics.
- Cost clarity block: fee structure, minimums, no-hidden-fees statement.
- Trust block: SIPC, SEC-registered, safeguards.
- Calculator: illustrative growth projection with clear "not guaranteed" disclosure.
- Final CTA with what happens after signup.

## Review notes
- Risk disclosures visible, not linked-to-footer.
- Calculator computes correctly and includes disclaimer.
- Language avoids jargon (Sharpe ratio, alpha, etc.) or defines it inline.
- No red-flag dark patterns (fake countdowns, urgency copy).
