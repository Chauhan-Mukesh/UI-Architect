# Pricing Variant 01 — Tier Comparison Cards

## Structure
- Header: title, subtitle, billing toggle (if applicable).
- Row of tier cards: name, price, cadence, short pitch, feature bullets, CTA.
- Below: FAQ addressing objection points (upgrade path, cancellation, taxes).

## Use when
- 2–4 plans with clearly different targets (starter, team, business, enterprise).

## Grid & spacing
- Cards equal width, equal height on desktop.
- Recommended tier is elevated (subtle border or lift), not neon.

## Breakpoints
- Mobile: cards stack in order (starter → recommended first, or in ascending price).
- Desktop: horizontal row.

## Accessibility
- Each tier is a landmark region with a labeled heading.
- Toggle for billing cadence updates prices via `aria-live` when values change.

## Anti-patterns
- Recommended tier so loud it distorts the comparison.
- Feature bullets that are the same across tiers ("email support") without qualifiers.
