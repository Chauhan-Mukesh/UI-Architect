# Pricing Variant 03 — Usage-Based / Calculator

## Structure
- Header explaining the pricing model in one paragraph.
- Interactive calculator: inputs for usage units, live output for cost.
- Tier or discount thresholds visualized.
- Contact for volume / enterprise below the calculator.

## Use when
- API products, cloud services, per-seat + usage hybrids.

## Grid & spacing
- Calculator gets a prominent card with input controls and computed output.
- Reset and copy-quote actions visible.

## Breakpoints
- Mobile: inputs stack vertically; output stays in view.
- Desktop: inputs left, output right.

## Accessibility
- Inputs are labeled; ranges announce values.
- Computed output updates via `aria-live="polite"` (not per keystroke).
- Currency and units always visible with the number.

## Anti-patterns
- Calculators that hide inputs behind an "advanced" toggle by default.
- Missing example presets for common usage.
