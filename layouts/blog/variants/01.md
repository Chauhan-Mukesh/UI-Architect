# Blog Variant 01 — Editorial Index

## Structure
- Top: featured/hero post with large image, title, dek, byline.
- Below: 2–4 secondary posts in a mixed grid.
- Further down: standard list with category chips.
- Sidebar (optional): about, newsletter, popular posts.

## Use when
- Publication-style blogs with editorial curation.
- Multiple category priorities.

## Grid & spacing
- Feature spans full content width.
- Secondary grid alternates sizes for editorial rhythm.

## Breakpoints
- Mobile: linear list.
- Desktop: editorial grid.

## Accessibility
- Hero post uses `<article>` with a heading link.
- Category chips are links, not decorative badges.

## Anti-patterns
- Rotating carousel of featured posts.
- Newsletter block dominating the first fold.
