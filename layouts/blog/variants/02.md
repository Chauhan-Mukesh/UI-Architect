# Blog Variant 02 — Clean List Index

## Structure
- Header with title, description, filter/tag row.
- Vertical list of posts: title, date, dek, category.
- Optional pagination or "load more".

## Use when
- Small to medium blog; the value is the writing, not the design.
- Reader expects chronological or category browsing.

## Grid & spacing
- List width 65–75ch max.
- Consistent spacing between items.

## Breakpoints
- Mobile: same list, slightly tighter typography.
- Desktop: same list; optional sidebar with about/subscribe.

## Accessibility
- Post list is a `<ol>` or `<ul>` if order is meaningful.
- Each item is an `<article>` with heading.

## Anti-patterns
- Infinite scroll without a fallback pagination link.
- Missing dates or hidden authorship.
