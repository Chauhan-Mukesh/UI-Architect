# Admin Variant 02 — Settings

## Structure
- Left sub-nav of settings groups (account, workspace, integrations, billing, security).
- Right side: setting cards, one section per group, each with form controls and inline help.
- Save behavior: per-section save, autosave, or explicit save — pick one and be consistent.

## Use when
- Configuration surfaces for accounts, workspaces, or apps.

## Grid & spacing
- Cards with labeled sections; consistent spacing between fields.
- Inline help below labels, warnings inline where they apply.

## Breakpoints
- Mobile: sub-nav collapses to a dropdown; sections stack.
- Desktop: two-region layout.

## Accessibility
- All form controls have labels.
- Save state feedback is announced (`aria-live="polite"`).

## Anti-patterns
- Silent saves without any feedback.
- Modal edit dialogs for every field.
