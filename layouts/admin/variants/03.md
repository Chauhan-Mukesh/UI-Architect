# Admin Variant 03 — Audit / Activity Log

## Structure
- Filter row: time range, actor, event type, resource.
- Main: chronological log with expandable rows for detail.
- Export and share controls.

## Use when
- Auditing, compliance surfaces, activity feeds.

## Grid & spacing
- Rows dense; timestamps prominent.
- Expandable detail with diff or context payload.

## Breakpoints
- Mobile: filters as a bottom sheet; rows show top-level info only.
- Desktop: full row visible.

## Accessibility
- Time in `<time datetime>` with human-readable local time and machine ISO.
- Expandable rows use `<details>` or `aria-expanded`.

## Anti-patterns
- Log with no permalinks for individual entries.
- Filters that don't sync to URL (impossible to share a view).
