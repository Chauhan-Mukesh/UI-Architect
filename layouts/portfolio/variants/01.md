# Portfolio Variant 01 — Editorial Grid

## Structure
- Short intro at top: name, discipline, one-line positioning.
- Grid of selected work (3–8 items) with title, role, and year on hover or below the thumbnail.
- Simple about + contact at the bottom.

## Use when
- Small, high-curation portfolio.
- Work is visually strong enough to speak without headline copy.

## Grid & spacing
- Asymmetric grid allowed: some items span 2 columns, some 1.
- Generous margins around the grid; page feels editorial.

## Breakpoints
- Mobile: single column with preserved order.
- Tablet: 2 columns.
- Desktop: 3–4 columns with occasional feature-span items.

## Accessibility
- Each thumbnail link has an accessible name that includes title and role.
- Grid order in DOM matches visual reading order.

## Anti-patterns
- Uniform card grid identical to every SaaS "features" section.
- Overlay text on imagery without contrast checking.
