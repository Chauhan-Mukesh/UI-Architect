# Portfolio Variant 03 — Case-Study Index

## Structure
- Header with intro + filter/tag row.
- List or grid of case studies with filterable tags (industry, discipline, year, role).
- Search when the collection is large enough to warrant it.
- Contact and about accessible from persistent nav.

## Use when
- Studio or freelancer with a substantial body of work.
- Different audiences need different subsets.

## Grid & spacing
- Filter row sticky on scroll if it improves navigation.
- Consistent card structure — thumbnail, title, role, tags.

## Breakpoints
- Mobile: filter as a bottom sheet or dropdown.
- Desktop: filter row with visible tag chips.

## Accessibility
- Filter controls are real form controls with labels.
- Result count announced via `aria-live="polite"`.

## Anti-patterns
- Filters that reload the page.
- Tags that look like clickable filters but aren't.
