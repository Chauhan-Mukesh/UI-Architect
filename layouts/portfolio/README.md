# Portfolio Layouts

## Purpose
Present selected work so the right audience can evaluate the maker and reach out.

## When to use which variant
- **01 Editorial grid** — curated selection where each piece is a case study.
- **02 Long-scroll narrative** — a single scroll telling a personal or studio story.
- **03 Case-study index** — for portfolios with many pieces where filtering matters.

## Universal rules
- Curate to 3–8 pieces on the index. If more, use filters, not more thumbnails.
- Case-study depth carries the pitch. Index pages stay quiet.
- Contact is one action. Not three.

## Anti-patterns
- Auto-playing videos in the grid.
- Thumbnails that don't reflect the finished work.
- Corporate marketing sections leaking into a portfolio.

## Accessibility
- Thumbnails wrapped in `<a>` with descriptive text (title + role).
- Focus states clearly visible against portfolio imagery.

## Responsive
- Mobile: grid to single column; retain hierarchy.
- Desktop: allow bespoke grid (spans, asymmetry).
