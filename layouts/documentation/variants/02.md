# Documentation Variant 02 — Two-Column

## Structure
- Left: site nav.
- Right: article content with in-page anchors on headings.
- Top: search, version switcher.

## Use when
- Guides, conceptual docs, tutorials that read linearly.
- No need for a persistent TOC.

## Grid & spacing
- Left nav 240–280px; content max-width around 720–800px.

## Breakpoints
- Mobile: nav drawer; content full-width.
- Desktop: two-column.

## Accessibility
- Site nav marked as `<nav aria-label="Documentation">`.
- Prev/next links at the end of each article.

## Anti-patterns
- Removing nav on desktop to save space.
- Hiding anchor links entirely — keyboard users need them.
