# Travel — Conversion Goals

## Primary conversion
- Target: booking completion and itinerary planning actions.
- Primary CTA: Book now / Plan trip.

## Secondary conversion
- Save-and-return path for users not ready to commit.
- Assisted support path for complex decision cases.

## KPI orientation
- Primary CTA intent rate
- Completion rate of primary flow
- Step drop-off analysis
- Confidence indicators after submission/booking
