# Legal — Layout Preferences

## Recommended structure
1. Core value proposition + high-intent CTA
2. Offer/listing/service highlights
3. Trust and risk-reduction content
4. Process or decision guidance
5. FAQs and objection handling
6. Final conversion block

## Layout rules
- Support scan-friendly comparison and prioritization.
- Keep mobile hierarchy optimized for quick decisions.
- Avoid noisy sections that compete with the primary action.
