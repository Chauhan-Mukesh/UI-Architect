# Restaurant — Common Components

## Core components
- Hero with signature imagery (dish, room, or moment) and primary reservation CTA
- Quick-facts strip: hours, address, phone, cuisine
- Menu section or link (structured, not a PDF-only path)
- Reservation widget or clear reservation link
- Gallery of dishes / room / events
- Story / chef / team block for identity
- Location map and directions
- Private events / catering path when applicable
- Reviews or press strip when strong

## Component expectations
- Every component works on mobile in one thumb.
- Menus load fast; big PDF-only menus are a failure.
- Photos are original where possible.
