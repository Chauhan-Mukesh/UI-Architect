# Restaurant — Layout Preferences

## Recommended structure
1. Hero (image + name + primary CTA)
2. Quick facts (hours, address, phone)
3. Menu (or clear menu entry)
4. Reservation
5. Story / identity
6. Gallery
7. Location / map
8. Footer with full contact and social

## Layout rules
- Imagery gets full-bleed or near-full-bleed treatment.
- Text overlays on imagery must maintain WCAG contrast.
- Mobile shows the same intent order; no hidden nav.
- Do not use decorative hero videos that block or delay reservation CTA.
