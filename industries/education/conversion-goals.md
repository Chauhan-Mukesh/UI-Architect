# Education — Conversion Goals

## Primary conversion
- Target: program inquiries and enrollment actions.
- Primary CTA: Apply now / Request information.

## Secondary conversion
- Save/continue actions for non-ready users.
- Assisted path (contact/support/advisor) for high-friction decisions.

## KPI orientation
- Primary CTA intent rate
- Form/service completion rate
- Drop-off by journey step
- Post-conversion confidence indicators
