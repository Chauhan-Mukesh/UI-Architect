# Construction — Example Direction

## Example page goals
- Enable project owners, procurement teams, and site managers to complete quote requests and project consultation bookings quickly.
- Guide users through one primary and one assisted conversion path.

## Suggested example scenario
- Build a focused conversion page with value hero, proof, process, FAQ, and closing CTA.
- Include mobile behavior and accessibility checkpoints in the brief.

## Review checklist
- Is the primary action always visible and clear?
- Is trust sufficiently established before commitment?
- Are readability and accessibility preserved across breakpoints?
