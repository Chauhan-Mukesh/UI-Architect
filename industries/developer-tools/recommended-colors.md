# Developer Tools — Recommended Colors

## Palette direction
- Base direction: dark/light developer-friendly palettes with clear state accents.
- Reserve accent intensity for CTA and status signaling.
- Enforce WCAG 2.2 AA contrast across text and controls.

## Usage constraints
- Do not encode status meaning by color alone.
- Keep interactive states visually distinct and consistent.
- Validate contrast in all core states and themes.
