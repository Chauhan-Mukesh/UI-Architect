# Insurance — Recommended Colors

## Palette direction
- Base direction: stable base palette with clear action/status semantics.
- Use accent colors primarily for action and state communication.
- Maintain WCAG 2.2 AA contrast for text and controls.

## Usage constraints
- Never rely on color alone for status meaning.
- Keep error/warning/success semantics consistent across pages.
- Validate contrast for hover, focus, and disabled states.
