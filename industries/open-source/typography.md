# Open Source — Typography

## Typographic style
- Tone target: clear, technical, unpretentious.
- Sans-serif body face pairs with a good monospace for code.
- Monospace choice is a signature; pick one and use it consistently.

## Readability rules
- Body copy 65–75ch; code blocks can extend wider.
- Monospace tabular-nums for API tables and metrics.
- Ligatures optional in code — off by default, opt-in via user setting if provided.
