# Open Source — Conversion Goals

## Primary conversion
- Target: installation, star, or first-successful-use.
- Primary CTA: "Get started" / install command (copyable).

## Secondary conversion
- Read the docs.
- Star / watch the repo.
- Join community channel.
- Sponsor / donate.

## KPI orientation
- Quick-start command copies
- Docs page-throughs (especially getting-started completion)
- GitHub star velocity from the site
- Community signup rate
