# Portfolio — Typography

## Typographic style
- Tone target: personal, deliberate, legible.
- Type is a chance to signal craft; a distinctive but readable pairing works well.
- One display + one text face, or a single well-chosen family across roles.

## Readability rules
- Body copy 60–75ch; case-study copy earns more room.
- Avoid overly light weights on off-white backgrounds.
- Headings can be expressive; body must not be.
