# Portfolio — Layout Preferences

## Recommended structure
1. Intro / positioning line
2. Selected work (largest content weight)
3. About / process / values
4. Contact

## Layout rules
- Break the grid; portfolios are one of the few places where restraint and asymmetry both work.
- Whitespace is content. Do not fill it.
- Mobile compresses but preserves case-study hierarchy.
- Avoid multi-section marketing scaffolding — this is not a landing page.
