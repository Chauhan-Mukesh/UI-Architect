# Automotive — Common Components

## Core components
- Hero with explicit value and CTA
- Capability/feature modules with scannable details
- Process timeline or step-by-step flow
- Trust/evidence sections (credentials, proof, metrics)
- FAQ and objection handling
- Conversion form with progressive fields

## Component expectations
- Each component must support a concrete decision point.
- Accessibility, error states, and focus behavior are mandatory.
