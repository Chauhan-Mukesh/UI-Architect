# Agency — Common Components

## Core components
- Editorial hero with statement-of-position, not a value bullet list
- Selected work grid or scroll-through with rich imagery and one-line context
- Case study cards linking to full case-study pages
- Services block written as capability areas, not deliverables
- Team/culture module (people, not stock photos)
- Awards/press strip when relevant to positioning
- Contact/brief-submission form with qualifying fields

## Component expectations
- Imagery is the primary content — treat it with layout weight.
- Hover and reveal states must feel intentional, not templated.
- Case study detail pages carry the depth; index pages stay curated.
