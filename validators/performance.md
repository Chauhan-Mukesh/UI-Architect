# Performance Validator

## Scope
Loading behavior, rendering efficiency, and interaction responsiveness.

## Pass criteria
- Prioritizes critical content and avoids render-blocking excess.
- Minimizes JS for non-essential behaviors.
- Uses optimized media strategy and sensible defaults.
- Prevents layout shift in key content regions.

## Severity model
- Critical: major delays or unstable rendering in primary flow.
- Major: avoidable performance regressions affecting engagement.
- Minor: optimization opportunities with low user impact.
