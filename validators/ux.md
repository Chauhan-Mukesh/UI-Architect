# UX Validator

## Scope
Task clarity, hierarchy, trust signals, and conversion readiness.

## Pass criteria
- Primary user goal is obvious above the fold.
- CTA strategy is singular and consistent.
- Evidence/proof appears before high-commitment actions.
- Content sequence matches user decision flow.

## Severity model
- Critical: user cannot complete primary task.
- Major: friction likely to reduce conversion significantly.
- Minor: polish opportunities without journey failure.
