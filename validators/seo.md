# SEO Validator

## Scope
Metadata completeness, semantic structure, and discoverability basics.

## Pass criteria
- Unique title and meta description aligned to intent.
- Proper heading hierarchy and semantic sectioning.
- Structured data considered where relevant.
- Link text clarity and crawl-friendly information order.

## Severity model
- Critical: missing metadata/structure harming indexability.
- Major: weak semantics reducing search relevance.
- Minor: enhancements for richer snippets and clarity.
