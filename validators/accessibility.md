# Accessibility Validator

## Scope
WCAG 2.2 AA alignment across structure, interactions, and content.

## Pass criteria
- Keyboard operability for all primary interactions.
- Visible and consistent focus behavior.
- Sufficient contrast across text and UI states.
- Screen-reader compatibility on primary journeys.

## Severity model
- Critical: inaccessible core task path.
- Major: substantial barriers in secondary interactions.
- Minor: improvements that increase inclusivity.
