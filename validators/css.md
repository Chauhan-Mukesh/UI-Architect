# CSS Validator

## Scope
Architecture, responsiveness, maintainability, and accessibility support.

## Pass criteria
- Uses tokens/variables for repeatable values.
- Responsive behavior preserves hierarchy and usability.
- Focus states and interactive states remain visible.
- Avoids inline style dependency and specificity escalation.

## Severity model
- Critical: breaks layout/accessibility in common viewports.
- Major: inconsistent patterns reducing maintainability.
- Minor: style hygiene and consistency issues.
