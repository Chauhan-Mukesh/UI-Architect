# Cognitive Load

## Objective
Reduce mental effort required to understand and complete tasks.

## Strategies
- Minimize simultaneous decisions per step.
- Chunk complex information into digestible units.
- Use familiar UI patterns for common actions.
- Replace long explanations with progressive reveal.
- Pre-fill, suggest, or infer where safe.

## High-load signals
- Multiple competing labels for the same concept.
- Long uninterrupted form fields without grouping.
- Repeated context switching between sections.

## Quality target
Users should complete primary flow without re-reading instructions more than once.
