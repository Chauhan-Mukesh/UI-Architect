# Gestalt

## Objective
Apply visual perception laws so users instantly understand grouping, priority, and relationships.

## Rules
- **Proximity**: related items stay visually near each other.
- **Similarity**: shared styles imply shared behavior.
- **Continuity**: align elements to create natural scanning paths.
- **Figure-ground**: maintain clear foreground/background separation.
- **Common region**: use containers to define conceptual groups.

## Validation checklist
- Are grouped items consistently spaced?
- Do visual styles map to interaction semantics?
- Is section flow obvious without reading all text?
- Is contrast sufficient to separate foreground content?
