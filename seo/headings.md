# SEO — Headings

## Purpose
Headings are the skeleton of the page. Search engines and screen readers both use them.

## Rules
- Exactly one `<h1>` per page.
- Headings never skip levels (`<h2>` → `<h4>` is a bug).
- Every `<section>` has a heading.
- Match `<title>` intent to `<h1>` intent (not necessarily identical text).
- Do not style non-headings as headings.

## Rules
- Heading text answers "what is this section about?"
- Do not use headings for microcopy (labels, badges).
- Do not stuff keywords into headings.

## Anti-patterns
- Multiple `<h1>`s.
- Skipped heading levels.
- Empty headings for visual space.
- Marketing bullet lists disguised as `<h3>`s.
