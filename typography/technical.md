# Typography — Technical

## Purpose
Typographic voice: **precise, honest, developer-friendly**.

## Pairing
- Display: Geometric or humanist sans.
- Body: High-legibility sans + a strong monospace for code.

## Rules
- Tabular numerals in tables
- Monospace with clear l/1/I/i distinction
- Ligatures in code are optional and off by default
- Body copy stays at 60–75ch measure.
- Use tabular numerals for tables and data.
- Fallback stack matches metrics of the primary.

## Anti-patterns
- Decorative display faces
- Monospace used for prose
- All-caps body copy.
- Ornamental script for anything beyond a headline.

## When to choose this voice
- Match against the tone adjectives in the design direction card.
- Cross-check with the industry recommendation before committing.
