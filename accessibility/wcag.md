# Accessibility WCAG

## Standard
Target WCAG 2.2 AA compliance for all generated interfaces.

## Required checks
- Perceivable: text alternatives, contrast, adaptable content.
- Operable: full keyboard access, visible focus, timing controls.
- Understandable: predictable navigation, input assistance, clear language.
- Robust: semantic structure compatible with assistive technologies.

## Evidence expected
- Check-by-check pass/fail notes.
- Violations mapped to impacted components.
- Remediation priority for any failed criterion.
