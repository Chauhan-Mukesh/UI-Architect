# Accessibility Keyboard

## Goal
All interactive functionality must work without a mouse.

## Mandatory behaviors
- Logical tab order matching visual reading order.
- No keyboard trap in dialogs, menus, or overlays.
- Enter/Space behavior aligned with native expectations.
- Escape closes dismissible overlays where appropriate.

## Validation
- Navigate full primary journey with keyboard only.
- Confirm all actionable controls are reachable and operable.
